1. Project Overview
ClassTrack is a console-based Java application for Student Attendance Management. It features Role-Based Access Control (RBAC) for HOD, Faculty, and Students.

2. Project Structure
The project is organized efficiently:

src/: Java source code (.java) in packages (classtrack, classtrack.hod, etc.)
bin/: Compiled bytecode (.class) - auto-generated.
lib/: External libraries (mysql-connector-j-8.3.0.jar).
setup.sql: Database initialization script.

3. How to Run
Prerequisites
Java JDK installed.
MySQL Server running.
Database reset (if needed) using setup.sql

Commands
Open a terminal in the project folder:

cmd
cd "c:\Users\Bigal\OneDrive\Documents\Aditya university\Projects\ClassTrack"

1. Compile (one-time or after code changes):
cmd

2. Run the Application:
cmd
java -cp "bin;lib/mysql-connector-j-8.3.0.jar" classtrack.MainMenu
